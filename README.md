***Deep-Dive Analysis & Interactive Dashboard Project
Task Overview***

This Task focuses on performing a deep-dive data analysis on a student dataset and building an interactive dashboard to answer important analytical questions.

The goal of the Task is to identify key insights about student demographics, course enrollment patterns, and academic distribution.
***Key Performance Indicators (KPIs)***

The following KPIs were defined to analyze the dataset:

Total Students
Measures the total number of enrolled students.

Gender Distribution
Shows the percentage of male and female students.

Most Popular Course
Identifies the course with the highest student enrollment.

Students per Academic Year
Shows how students are distributed across academic years.

***Deep-Dive Analysis***

A deep-dive analysis was performed on course enrollment patterns and academic year distribution.

This analysis helps answer key questions such as:

Which course has the highest number of students?

How are students distributed across academic years?

***Dashboard***

An interactive dashboard was built using Power BI to visualize the insights from the dataset.

The dashboard includes:

***KPI cards***

Course enrollment analysis

Gender distribution visualization

Academic year distribution

Tools Used

Python (Pandas, Matplotlib, Seaborn)

Power BI
